# Synthesis Directory

This directory stores synthesis-related scripts, reports, and generated netlists.

---

# Synthesis Goals

- Convert RTL to gate-level netlist
- Analyze timing
- Estimate area
- Validate synthesizability

---

# Expected Files

| File Type | Description |
|---|---|
| .v | Synthesized netlist |
| .rpt | Timing and area reports |
| .log | Synthesis logs |

---

# Synthesis Toolchain

- Yosys
- OpenROAD
- OpenSTA
