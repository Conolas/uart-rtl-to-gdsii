# Physical Design Visualizations

This directory contains visual representations of the RTL-to-GDSII flow generated during ASIC implementation.

## Visualization Categories

### Synthesis
- Gate-level schematics
- Flattened netlist visualizations

### Floorplan
- Core layout
- PDN structure
- Macro placement

### Placement
- Standard cell placement
- Density maps
- Placement optimization

### CTS
- Clock tree layouts
- Clock buffer distribution

### Routing
- Global routing
- Detailed routing
- Congestion maps

### Final
- Final GDSII layouts
- IR-drop analysis
- Final routed chip views

## Visualization Tools

- KLayout
- OpenROAD GUI
- Yosys Graphviz generation

## Purpose

These visualizations document the complete transformation of the UART design from RTL logic into a fabrication-ready physical silicon layout.
