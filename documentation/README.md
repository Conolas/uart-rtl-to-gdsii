# Documentation

This directory contains detailed documentation related to the complete RTL-to-GDSII implementation flow of the UART ASIC project.

## Contents

- OpenROAD installation and setup guide
- RTL-to-GDSII flow explanation
- Physical design documentation
- ASIC backend concepts
- Flow debugging notes
- Error resolutions and troubleshooting
- Design methodology and implementation details

## Purpose

The goal of this documentation is to provide a complete reference for:

- Beginners learning ASIC design
- Students exploring physical design
- Engineers experimenting with open-source EDA tools
- Understanding the complete backend flow from RTL to GDSII

## Tools Covered

- OpenROAD
- Yosys
- KLayout
- GTKWave
- Icarus Verilog
- Docker
- SKY130 PDK
