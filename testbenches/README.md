# Testbench Directory

This directory contains verification testbenches for all UART RTL modules.

---

# Testbenches

| File | Description |
|---|---|
| baud_gen_tb.v | Verifies baud pulse generation |
| uart_tx_tb.v | Verifies UART transmission |
| uart_rx_tb.v | Verifies UART reception |
| uart_top_tb.v | End-to-end UART loopback verification |

---

# Verification Features

- Clock generation
- Reset sequencing
- UART stimulus injection
- Loopback verification
- Functional validation
- Waveform dumping for GTKWave

---

# Simulation Tools

- Icarus Verilog
- GTKWave
